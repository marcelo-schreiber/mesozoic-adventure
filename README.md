## Instalação

Instale [Python](https://www.python.org/) e em um terminal rode:
```bash
pip install pygame
```
depois, para iniciar o jogo:

```bash
python main.py
```

se não funcionar, você pode trocar o 'python' por 'python3'.
Caso queira gerar um novo .exe, verifique o arquivo 'settings.py'.
